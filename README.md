# ToyProject(Board,Signin)

## Introduce
- Spring MVC 기본 게시판 및 로그인,회원가입 구현

## 기술 도구
- Java17
- Springboot 3.2.3
- MyBatis, MySqlworkbench
- lombok
- Thymeleaf, bootstrap
- HTML, CSS, JavaScript
